Event

A generic message type that act's as a wrapper for all system events. When an event is `Emit`'ed by a producer

the `Event` is stored and forwarded to the correct consumer.

FieldTypeLabelDescription
id string

id - is a uuid to identify each event of the system

aggregate_id string

aggregate_id - is the identifier of the aggregate the `Event` relates to.

transaction_id string

transaction_id - is a uuid for each transaction that can be used to string together many differnt events to one executed command

data string

data - the `data` payload of the event system that can be of any message type the consumer will only be interested in specific types of `Events` from a specifc source. the `event_type` is used as the deserialization/serialization as the type identifier of the `data` value.

created_at google.protobuf.Timestamp

The datetime when the event was stored

published_at google.protobuf.Timestamp

The datetime when the event was published to it's event stream for processing

aggregate_type AggregateType

event_code EventCode

side_affect bool

used to determin reply strategy
