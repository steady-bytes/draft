## dctl config

Initialize the dctl configuration

### Synopsis

Initialize the dctl configuration. Be sure to run
this from the root of the draft repository

```
dctl config [flags]
```

### Options

```
  -h, --help   help for config
```

### Options inherited from parent commands

```
      --config string   config file (default is $HOME/.dctl.yaml)
```
